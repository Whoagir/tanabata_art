# Иконки башен (визуальные «скиллы» в панели информации)

Не логика, а подписи для игрока: тип атаки и механики (лазер, горение, замедление и т.д.).

- **attack/** — тип атаки: projectile.png, laser.png, aoe.png, beacon.png.
- **mechanics/** — доп. эффекты: slow.png, burn.png, poison.png (и при необходимости другие).

Placeholder'ы генерируются скриптом `scripts/generate_tower_icons.py`. Свои картинки — заменить файл с тем же именем.
