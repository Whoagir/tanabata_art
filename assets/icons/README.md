# Иконки игры

Иконки разложены по папкам, чтобы не путаться при росте проекта.

- **abilities/** — способности из `data/ability_definitions.json` (уклонение, блинк, рефлекшн и т.д.). Используются для врагов и для любых сущностей с массивом abilities. Placeholder'ы: `scripts/generate_ability_icons.py`.

- **towers/** — визуальные «скиллы» башен для панели информации (тип атаки + механики): лазер, снаряд, AoE, маяк, замедление, горение, яд и т.д. Подпапки: `attack/` (тип атаки), `mechanics/` (доп. эффекты). Placeholder'ы: `scripts/generate_tower_icons.py`.

- **enemies/** — зарезервировано под иконки врагов (портреты, типы и т.д.). При необходимости можно завести подпапки по типам врагов.

Любые PNG можно заменить своими картинками с тем же именем файла.
