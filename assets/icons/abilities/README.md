# Иконки способностей (abilities)

Способности из `data/ability_definitions.json` — используются для **врагов** (и при необходимости для башен с полем abilities). Файлы: `{id}.png` (evasion.png, blink.png и т.д.).

**Placeholder'ы** сгенерированы скриптом `scripts/generate_ability_icons.py`. Чтобы пересоздать или добавить новые способности в JSON и заново сгенерировать заглушки:

```bash
# из корня проекта (WSL)
python3 scripts/generate_ability_icons.py
```

**Свои картинки:** просто замените любой PNG файл с тем же именем (например, evasion.png) — в игре подставится ваша текстура. Размер по желанию (рекомендуется квадрат 64x64 или 128x128).

`unknown.png` — fallback для неизвестной способности (если файла по id нет).
